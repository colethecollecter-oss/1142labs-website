### Deployment Instructions

1. Copy all files to your web server
2. Ensure `/assets/` directory is accessible
3. Ensure `/breakthroughs/` directory is accessible
4. Update contact form backend if needed
5. Configure CORS if needed for external resources

### Customization

- Colors: Modify CSS variables in `:root`
- Fonts: Update font imports in `<head>`
- Logo: Replace image files in `/assets/`
- Content: Edit HTML sections directly
- Animations: Adjust keyframe timing in CSS
