CUSTOMIZATION GUIDE
================================================================================

COLORS:
Edit /assets/enhanced-style.css, find :root section:
- --color-primary-neon: #00F0FF (cyan)
- --color-secondary-neon: #FF10F0 (pink)
- --color-foundation: #0D1117 (dark)
- --color-warning: #FF4136 (red)
- --color-active: #4ADE80 (green)

FONTS:
Edit <head> section in index.html:
- Primary: Space Grotesk (sans-serif)

LOGO:

CONTENT:

ANIMATIONS:

- Adjust timing values as needed

TROUBLESHOOTING
================================================================================

IMAGES NOT LOADING:
- Check file paths are correct
- Verify /assets/ directory exists
- Check file permissions (644 for files, 755 for directories)
- Ensure web server can read files

STYLES NOT APPLYING:
- Clear browser cache (Ctrl+Shift+Delete)
- Check CSS file path in <head>
- Verify enhanced-style.css is uploaded
- Check browser console for errors

VIDEOS NOT PLAYING:
- Verify featured-video.mp4 exists
- Check video codec compatibility
- Ensure MIME type is set to video/mp4
- Test in different browsers

RESPONSIVE ISSUES:
- Check viewport meta tag in <head>
- Test with browser dev tools
- Clear cache and reload
- Check CSS media queries

CONTACT FORM NOT WORKING:
- Form is HTML only - needs backend setup
- Implement PHP, Node.js, or other backend
- Set form action to your endpoint
- Test form submission
